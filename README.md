# passvault

## install
```bash
$ git clone https://github.com/ritarock/passvault.git
$ cd passvault
$ make install
```

## Usage
```bash
$ passvault help    
generate password cli

Available Commands:
  generate: generate_command generates a password
  list:     list_command gets generated list
  get:      get_command get a generated password

$ passvault generate help
generate password

Usage:
  generate <title> <url>

$ passvault get help
get a generated password
The retrieved password is saved to the clipboard.

Usage:
  get <id>
```
